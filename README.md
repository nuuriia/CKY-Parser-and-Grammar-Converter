# CKY-Parser-and-Grammar-Converter
Implementation of the Cocke–Younger–Kasami (CKY) algorithm for context-free grammar parsing. Features a CFG to Chomsky Normal Form (CNF) converter and a Probabilistic CKY (PCKY) version for statistical parsing. Developed for "Programació i Algorismia Avançada" (UPC - FIB).
